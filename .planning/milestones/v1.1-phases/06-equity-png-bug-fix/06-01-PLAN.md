<objective>
Fix blank equity curve PNGs during /brrr:execute iterations by returning trades and equity_curve from run_backtest() and using them directly in the execute.md PNG generation template.

Purpose: Users currently see blank/empty PNGs after each backtest iteration, making visual analysis impossible. The root cause is that compute_all_metrics() returns only 9 scalar metrics -- the equity_curve array computed inside run_backtest() is passed to compute_all_metrics() but discarded from the return value.

Output: Two modified reference files that produce visible equity curve PNGs when trades exist and gracefully skip PNG generation when no trades occur.
</objective>
