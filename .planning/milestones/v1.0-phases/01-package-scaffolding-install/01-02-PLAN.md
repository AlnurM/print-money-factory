  <behavior>
**Individual metric functions:**

1. `sharpe_ratio(returns, risk_free_rate=0.0, trading_days=252) -> float`
   - Known answer: 252 days of alternating +1%/-0.5% returns -> Sharpe between 3.5 and 4.0
   - Constant returns (zero std) -> NaN
   - Empty array -> NaN
   - Single element -> NaN

2. `sortino_ratio(returns, risk_free_rate=0.0, trading_days=252) -> float`
   - Known answer: same alternating returns -> Sortino > Sharpe (less downside)
   - All positive returns (no downside) -> NaN
   - Empty array -> NaN

3. `calmar_ratio(returns, equity_curve, trading_days=252) -> float`
   - Known answer: annualized_return / abs(max_drawdown)
   - No drawdown (equity_curve monotonically increasing) -> NaN
   - Empty inputs -> NaN

4. `max_drawdown(equity_curve) -> float`
   - Known answer: [100, 120, 90, 110] -> -0.25 (25% drop from 120 peak)
   - Monotonically increasing -> 0.0
   - Single value -> 0.0

5. `win_rate(trades) -> float`
   - Known answer: 2 winners, 2 losers -> 0.5
   - Empty trades -> 0.0
   - All winners -> 1.0

6. `profit_factor(trades) -> float`
   - Known answer: gross_profit=200, gross_loss=100 -> 2.0
   - No losers -> inf (if any profit) or 0.0 (if no trades)
   - No winners -> 0.0
   - Empty trades -> 0.0

7. `expectancy(trades) -> float`
   - Known answer: trades with pnl [100, -50, 75, -25] -> 25.0
   - Empty trades -> 0.0

8. `trade_count(trades) -> int`
   - Returns len(trades)

9. `net_pnl(trades) -> float`
   - Sum of all trade pnl values
   - Empty trades -> 0.0

**Aggregate function:**

`compute_all_metrics(trades=None, equity_curve=None, daily_returns=None, trading_days=252, risk_free_rate=0.0) -> dict`
   - With trade log input: derives daily returns from trades, computes equity curve, returns all 9 metrics
   - With daily_returns input: computes equity curve from returns, returns ratio metrics (Sharpe, Sortino, Calmar, MaxDD) and NaN for trade-based metrics
   - Returns dict with keys: sharpe_ratio, sortino_ratio, calmar_ratio, max_drawdown, win_rate, profit_factor, expectancy, trade_count, net_pnl

- `sample_returns`: 252-day numpy array with known statistical properties
- `sample_equity`: Equity curve derived from sample_returns starting at 10000

Key implementation details for metrics.py:
- All functions accept numpy arrays, not pandas Series (conversion happens in compute_all_metrics if needed)
- `sharpe_ratio`: `mean(excess) / std(excess, ddof=1) * sqrt(trading_days)` where `excess = returns - risk_free_rate/trading_days`
- `sortino_ratio`: same but std computed only on negative excess returns
- `calmar_ratio`: `mean(returns) * trading_days / abs(min(drawdown))` where drawdown = (equity - running_max) / running_max
- `max_drawdown`: returns negative fraction (e.g., -0.25 means 25% drawdown)
- `profit_factor`: `sum(positive_pnl) / abs(sum(negative_pnl))`
- Trade log format: `[{'entry_time': str, 'exit_time': str, 'entry_price': float, 'exit_price': float, 'size': float, 'side': str, 'pnl': float, 'commission': float}]`
- `compute_all_metrics` with trades: computes equity curve as cumulative sum of pnl values starting from initial_capital (default 10000), derives daily returns from equity curve
- `compute_all_metrics` with daily_returns: computes equity from `initial_capital * cumprod(1 + returns)`, trade-based metrics return 0/NaN as appropriate
  </implementation>
