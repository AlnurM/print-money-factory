  <action>
Create `bin/install.mjs` -- the main install script invoked by `npx print-money-factory install`.

The file starts with `#!/usr/bin/env node` shebang.

**Imports (all Node.js stdlib -- zero external deps):**
```javascript
import { fileURLToPath } from 'url';
import { dirname, join, resolve } from 'path';
import { homedir, platform } from 'os';
import { existsSync, mkdirSync, copyFileSync, readdirSync, statSync, readFileSync, writeFileSync } from 'fs';
import { execSync } from 'child_process';
```

**Constants:**
```javascript
const __filename = fileURLToPath(import.meta.url);
const __dirname = dirname(__filename);
const PKG_ROOT = resolve(__dirname, '..');
const HOME = homedir();
const COMMANDS_DIR = join(HOME, '.claude', 'commands', 'brrr');
const PMF_DIR = join(HOME, '.pmf');
const VENV_DIR = join(PMF_DIR, 'venv');
const IS_WIN = platform() === 'win32';
const PYTHON_BIN = join(VENV_DIR, IS_WIN ? 'Scripts' : 'bin', IS_WIN ? 'python.exe' : 'python');
const PIP_BIN = join(VENV_DIR, IS_WIN ? 'Scripts' : 'bin', IS_WIN ? 'pip.exe' : 'pip');
```

**Helper: `copyDirRecursive(src, dest)`**
- Uses `mkdirSync(dest, { recursive: true })`
- Reads source entries with `readdirSync`
- For directories: recurse
- For files: `copyFileSync(srcPath, destPath)`
- Handles nested directories (e.g., references/pinescript-examples/)

**Helper: `detectPython()`**
- Tries candidates in order: `python3`, `python`, and on Windows also `py -3`
- For each candidate, runs: `{candidate} -c "import sys; v=sys.version_info; print(f'{v.major}.{v.minor}')"`
- Wraps in try/catch (execSync throws if command not found)
- Parses output, checks major >= 3 and minor >= 10
- If found, returns the candidate path string
- If none found, prints error:
  ```
  Error: Python 3.10+ is required but was not found.

  Install Python 3.10 or later:
    macOS:   brew install python@3.12
    Ubuntu:  sudo apt install python3.12 python3.12-venv
    Windows: https://www.python.org/downloads/

  Then run this command again.
  ```
  Then calls `process.exit(1)`

**Main function `main()`:**

1. Check `process.argv[2] === 'install'`. If not, print `Usage: npx print-money-factory install` and exit(1).

2. Print banner: `\nPrint Money Factory - Installing...\n`

3. **Copy commands/** -> `COMMANDS_DIR`:
   - `copyDirRecursive(join(PKG_ROOT, 'commands'), COMMANDS_DIR)`
   - Count files copied, print: `  [OK] Commands -> ~/.claude/commands/brrr/ (8 files)`

4. **Copy workflows/, templates/, references/** -> `PMF_DIR`:
   - For each of `['workflows', 'templates', 'references']`:
     - `copyDirRecursive(join(PKG_ROOT, dir), join(PMF_DIR, dir))`
     - Count files, print: `  [OK] {dir}/ -> ~/.pmf/{dir}/ ({N} files)`

5. **Write version file** at `join(PMF_DIR, '.version')`:
   ```javascript
   const pkg = JSON.parse(readFileSync(join(PKG_ROOT, 'package.json'), 'utf8'));
   writeFileSync(join(PMF_DIR, '.version'), JSON.stringify({
     version: pkg.version,
     installed: new Date().toISOString(),
     package: 'print-money-factory'
   }, null, 2));
   ```
   Print: `  [OK] Version file written (v${pkg.version})`

6. **Detect Python:**
   - `const pythonPath = detectPython();`
   - Print: `  [OK] Python found: {pythonPath} ({version})`

7. **Create or update venv:**
   - If `existsSync(VENV_DIR)` AND `existsSync(PYTHON_BIN)`:
     - Print: `\n  Updating existing venv...`
     - Run: `execSync(\`"${PIP_BIN}" install --upgrade -r "${join(PKG_ROOT, 'requirements.txt')}"\`, { stdio: 'inherit' })`
   - Else:
     - Print: `\n  Creating Python venv...`
     - Wrap venv creation in try/catch:
       ```javascript
       try {
         execSync(`"${pythonPath}" -m venv "${VENV_DIR}"`, { stdio: 'inherit' });
       } catch (e) {
         // Debian/Ubuntu ensurepip issue
         console.error('\nError: Failed to create venv. On Ubuntu/Debian, install:');
         console.error('  sudo apt install python3-venv');
         console.error('\nThen run this command again.');
         process.exit(1);
       }
       ```
     - Then: `execSync(\`"${PIP_BIN}" install -r "${join(PKG_ROOT, 'requirements.txt')}"\`, { stdio: 'inherit' })`

8. **Verify installation:**
   ```javascript
   try {
     execSync(`"${PYTHON_BIN}" -c "import pandas; import numpy; import ccxt; import yfinance; print('OK')"`, { stdio: 'pipe' });
     console.log('  [OK] Python dependencies verified');
   } catch (e) {
     console.error('\nWarning: Some Python dependencies failed to import. Run:');
     console.error(`  ${PYTHON_BIN} -m pip install -r ${join(PKG_ROOT, 'requirements.txt')}`);
   }
   ```

9. **Print summary:**
   ```
   Install complete!

   Commands installed:  ~/.claude/commands/brrr/
   PMF home:           ~/.pmf/
   Python venv:        ~/.pmf/venv/

   Get started: use /brrr:new-milestone to create your first strategy.
   ```

10. Call `main()` at the end of the file.

**Error handling throughout:** Every `execSync` call that can fail is wrapped in try/catch with actionable error messages. The script never shows a raw stack trace to the user.
